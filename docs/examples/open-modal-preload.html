You can click the button above to display the viewer.


In the current example, the modal/viewer element is always on the DOM, and the viewer starts loading when the page is ready. When the user clicks the button, the model is already preloaded, but this could add unwanted extra loading data for users that never click on the button.


Performance: When the viewer is used inside a modal, make sure it has width:0px and height:0px when the modal is hidden. This will prevent useless 3D computations.
